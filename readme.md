# Yark-AdminMS通用后台管理系统  
**主要技术**   
  
  
1. springboot  
2. shiro  
3. mybatisPlus  
4. layui  
  
  
**实现功能**  
  
  
1. 菜单管理  
2. 权限管理  
3. 用户管理   
4. 角色管理   
5. 部门管理    
6. 缓存管理  
7. 日志管理  
8. 数据库管理--使用druid  
9. 发布公告
